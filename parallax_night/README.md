# Parallax Night Webpage

Welcome to the **Parallax Night** project! 

This is a **weekend mini frontend project** dedicated to building an interactive layout using advanced CSS techniques and Vanilla JavaScript.

## 🚀 Features & Skills Showcased

- **Parallax Scrolling Effect:** Creating an immersive 3D illusion by moving foreground and background elements at different speeds based on keyboard/mouse scrolling.
- **Custom Scrollbar:** A stylized, custom-designed scrollbar that matches the dark, cohesive aesthetic of the webpage, overriding the browser's default scrollbar styling.
- **Dynamic Element Positioning:** Adjusting element coordinates mathematically using the scroll position.

## 🛠️ Technologies Used

- **HTML5:** Structuring the layers and content elements.
- **CSS3:** Flexbox/Grid layouts, absolute positioning, Z-index stacking, custom scrollbar styling (`::-webkit-scrollbar`), and responsive design.
- **Vanilla JavaScript:** Listening to the window's `scroll` events to dynamically manipulate the inline styles and calculate transform values based on `window.scrollY`.

## 📂 Project Structure

- `index.html` - The main structure.
- `style.css` - Custom styling and scrollbar styles.
- `script.js` - Parallax scroll logic.
- `img/` - Assets and layered images forming the parallax depth.
